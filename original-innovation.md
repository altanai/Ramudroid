# RamuDroid Original Innovation

## Innovator

Altanai Bisht

## Summary

RamuDroid is an autonomous outdoor garbage-collecting robotics innovation developed from 2014 to present, spanning multiple hardware and software generations. The project combines robotics, computer vision, IoT communication, and renewable power design to detect and collect litter in outdoor/public environments.

## Innovation Scope

- Hardware/software co-design across multiple robot versions
- Computer vision evolution from classical models to CNN-based detection
- Solar and battery hybrid power strategies
- Remote and near-edge control with streaming-based monitoring

## Version Evolution

| Version | Timeline | Highlights |
|---|---|---|
| v1.0 | 2016 | Initial frame, remote movement, manual brush control |
| v3.0 | 2017 | Camera streaming and web dashboard support |
| v5.0 | 2018 | Improved mechanics and sensor integration |
| v6.5 | 2019 | Better autonomous routing and garbage detection refinements |
| v7.0 (Surajdroid) | 2020 | Solar integration, autonomous triggers, obstacle avoidance |
| v8.0+ | 2021 onward | Improved CNN-based models and operational autonomy |

## Technical Highlights

- Raspberry Pi + Arduino split architecture for high/low-level control
- CV-triggered cleaning behavior to optimize energy usage
- Sensor fusion: camera, ultrasonic, IR, weight sensing
- Multi-network strategy: WiFi, BLE, LTE/4G (generation dependent)
- WebRTC/real-time media channels for remote observation
